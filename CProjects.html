File Sorter

This project comes with 3 programs, and this is how you use them:

Use RandomNumListGenerator.c to generate a .txt file consisting a list of random numbers in a range and amount of your choice.

Use SortingListFile.c to input that generated .txt file in order to create a NEW .txt file that contains the sorted list of those numbers.

Use SortingAlgorithms.c to generate a number list without a file and see it get sorted.

This program only works with .txt files that contain a random list of numbers. I made this to learn about the different sorting methods.
