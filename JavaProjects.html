Find the Ace of Spades Game

This project is the very first big project I made in Java GUI code. It is a made up card game and the rules are as follows:

Click on the facedown cards to reveal them. If a White Joker is shown, then the cards except the White Joker are shuffled and faced back down. If a Black Joker is shown, the game will end with a bad fortune. If the Ace of Spades is shown, you'll receive a tarot fortune.

Matrix Calculator

This calculator is specifically for Linear Algebra. After taking a Linear Algebra course, I realize that I had looked up so many different matrx calculators online to check my work. Therefore, I decided to make a matrix calculator, to calculate all of the different matrix-related problems that Linear Algebra students will run into. Here are what the Matrix Calculator calculates:

Addition, Subtraction, Multiplication, Row Echelon, Reduced Row Echelon, Inverse, Determinant, Transpose, Null Space, Column Space, Row Space, and Rank

Eigenvalues and Diagonalization will be added in the calculator soon due to their complexity. The Matrix Calculator for Android is still in progress. However, the Java GUI code for the Matrix Calculator is complete.

Ninja Shooting Game

After learning more about the Java GUI, I want to practice making a small game out of it by using the Java GUI, and so I try making this one. The rules of the game are as follows:

You have 30 seconds to hit the falling shapes. There are three different types of shapes with different values:

Blue Square: The slowest shape that is worth 5 points

Green Round Square: A shape that is worth 10 points

Red Circle: The fastest shape that is worth 20 points

Your score will be displayed when time runs out
